ENGINEERING TOOLKIT:
THE REMAINDER THEOREM

Se um polinômio P(x) é dividido pelo binômio (x - c), então o resto R dessa divisão é exatamente igual a P(c).

// Aplicando ao problema da imagem:

INPUT: P(3) = -2

// Interpretação do Engenheiro:
Isso significa que "c" é 3.
Logo, se dividirmos P(x) por (x - 3),
o resto SERÁ -2.

OUTPUT: Alternativa D está correta.

O Diferencial: Teoria que Ninguém Ensina

Essa é uma questão clássica do SAT que derruba brasileiros. Por que? Porque tentam "adivinhar" o polinômio ou fazer divisões longas, em vez de usar a definição teórica.

A Lógica do Engenheiro:

Eu não perco tempo. Eu vejo $P(3) = -2$ e imediatamente aciono o Teorema do Resto (explicado no painel técnico ao lado). A resposta D é a definição literal do teorema.

Por que A, B e C estão erradas? (O Teorema do Fator)

Para que um binômio $(x - c)$ seja um FATOR de um polinômio, a divisão deve ser exata. Ou seja, o resto deve ser ZERO.

Pelo Teorema do Resto, isso só acontece se $P(c) = 0$.

  • ❌ A) (x-5) é fator?
    Só seria verdade se P(5) = 0. O problema não disse isso.
  • ❌ B) (x-2) é fator?
    Só seria verdade se P(2) = 0. Não sabemos.
  • ❌ C) (x+2) é fator?
    Note que x+2 é x-(-2). Só seria verdade se P(-2) = 0.
